# Sorting Algorhithm Visualizer
A JavaScript sorting algorthihm visualizer that sorts from 1 to 100 with animations.

- Heap Sort
- Merge Sort
- Quick Sort
- Selection Sort
- Insertion Sort

## Tools
- JavaScript
- HTML/CSS
- Git
- Visual Studio Code
